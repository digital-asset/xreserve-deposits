# Canton depositToRemote UI

Single page ui for depositing USDC to Canton via the xReserve contract.

## Setup

1. Install dependencies:

   ```bash
   npm i
   ```

2. Configure `.env`:
   - Update contract addresses if needed

3. Get testnet USDC:
   - Visit <https://faucet.circle.com> to get testnet USDC on Ethereum Sepolia

## Running the UI locally

```bash
npm run dev
```
